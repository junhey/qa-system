# qa-system
仿 StackOverflow 开发在线问答系统

```

// 安装模块
pip3 install -r  pip_req/requirements_1.txt 

// 激活虚拟环境
$ source venv/bin/activate

// 初始化数据库的操作
python manage.py db init
python manage.py db migrate
python manage.py db upgrade


//运行服务器
python run.py

//访问 http://127.0.0.1:9000 ，点击「注册」按钮，注册用户名和密码，进行登录

```
